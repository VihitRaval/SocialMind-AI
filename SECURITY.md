# Security Policy

## Supported Versions

Only the latest release (v1.0.0) is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of **SocialMind AI** seriously. If you find any vulnerabilities or security concerns:

1. **Do not** open a public GitHub issue.
2. Please send a report via email to **vihit.raval@example.com** with details about the vulnerability.
3. Include clear steps to reproduce the issue, along with any relevant proof-of-concept code.

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to resolve the issue before public disclosure.
